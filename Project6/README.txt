How to compile and run this program:

This is a Java program.

To compile it, be in the src directory where the source code lives, and run "javac *.java".

To run the program, be in the src directory and use “java Project6 <path of the input .asm file>”. The program
accepts one command line argument, which is the path of the input file that you want to be translated.


Functionality:

This program is an assembler, which reads in a .asm file and translates it into a .hack file.
The output file <filename>.hack will be in the same directory and have the same filename as the input file.
